# lh-my-first-storybook

Este paquete me sirve para practicar React Componentes + Storybook + NPM Publish

```
npm install lh-my-first-storybook
```

```
yarn add lh-my-first-storybook
```

```
pnpm add lh-my-first-storybook
```
